# MessagerieC

Ce projet a été effecuté durant ma deuxième année d'école d'ingénieur par équipe de deux dans le cadre du cours Programmation Réseau.

L'objectif est de développer une messagerie (server/client) en C pour notamment :

-Contrôler les noms des utilisateurs qui se connectent au server (nom différent), possibilité de le changer

-échanger des messages privés et messages à plusieurs personnes

-créer des salons textuels  


J'y ai alors ajouté une interface, pour le client, avec la bibliothèque Ncurse pour avoir une zone d'affichage et une zone d'entrée des commandes et messages.
